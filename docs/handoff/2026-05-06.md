### 미완 — B-3 2단계 (다음 큰 작업)

대시보드의 실시간 구독 3개가 아직 Client SDK 사용:
- `subscribeToEvent` — 대시보드 이벤트 상태 구독
- `subscribeToHostEvents` — 대시보드 이벤트 목록 구독
- `subscribeToClips` — 대시보드 클립 목록 구독 (clips read는 이미 `if request.auth != null`이라 현재 대시보드는 정상)

B-3 2단계 완료 후:
- `events` read를 `if false`로 완전 잠금 (Admin SDK only)
- `clips` read도 대시보드 구독이 서버 이전되면 `if false`로 잠금 가능

**참고:** `clips` read는 현재 이미 `request.auth != null` — 대시보드(인증된 호스트)는 정상 동작 중. B-3 2단계 전까지 방치해도 기능 이상 없음.
